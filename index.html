Basically localStorage and sessionStorage both are two web storage objects for storing data on the client machine. But the main difference is localStorage data doesn't expire even closed the tab. On the other hand, sessionStorage data is cleared when the page session ends or closed page tab. localStorage is available in HTML 5 that allows storing data with no expiration date that is a type of web storage object. In contrast, sessionStorage available in HTML 5 allows storing data for one session that is a type of web storage object.
